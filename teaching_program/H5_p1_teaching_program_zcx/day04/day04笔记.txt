1、什么是css?	Cascading  Style  Sheet（层叠样式表）的缩写，用于控制网页外观。（修饰和美化html）
	优点：
	1>使网页代码更少，网页下载更快
	2>实现了内容与样式的分离，使网站维护更快捷
	3>使网页与浏览器的兼容性更好

2、	HTML	 超文本标记语言	从语义的角度描述页面结构。
	CSS	层叠式样式表		从审美的角度负责页面样式。
	JS	JavaScript		从交互的角度描述页面行为。
	
3、CSS就是由三个基本部分组成的——“对象”、“属性”和“值”。
	语法：
		选择器{
			属性1：属性值;
			属性2：属性值;
			... 
		}
	例如：
	h1{
		color:red;
		...
	}

4、CSS样式表分为4种：行内样式表、内嵌样式表、外链样式表、外部导入样式表。
	(1)、行内样式表（内联样式表、直进式样式表）--在html标记内，使用style属性定义css样式。
	如：	<p style=“属性:属性值;”>内容</p>
	设置不同的css样式，各个样式之间互不影响，分别显示自己的样式效果。
	
	(2)、内嵌样式表--将css添加到<head>与</head>之间，并用<style></style>标记声明的一种样式。	
	语法：
		<style  type="text/css">
			/*这里写css内容*/
			P{ 属性1：属性值1；属性2：属性值2}
			div{ 属性1：属性值1；属性2：属性值2}
		</style>
		
	(3)、外部链接样式表--在外部定义css样式表，通过<link/>链接标记链接到页面中的一种样式。
 	语法：
	<link rel="stylesheet" type="text/css" href="CSS文件地址" />
    1) link: 链接样式表
    2) href：指定了css样式表所在位置
    3) rel：指定链接到样式表，其值为stylesheet,不可省略
    4) type：表示样式表类型为css样式表

	(4)、外部导入样式表--在HTML文件初始化时，会被导入到HTML文件内，作为文件的一部分，类似于内嵌样式表效果。
	语法：	
	<head>
  		 <style type="text/css">
   			 @import url(css文件地址);
   		</style>
  	</head>

	(5)导入和链接式样式表优缺点比较:
	导入式会在整个网页装载完后再装载CSS文件，因此这就导致了一个问题，如果网页比较大则会儿出现先显示无样式的页面，
	闪烁一下之后，再出现网页的样式。这是导入式固有的一个缺陷。
	使用链接式在网页文件主体装载前装载CSS文件，因此显示出来的网页从一开始就是带样式的效果的，这是链接式的优点。
	
	(6)CSS样式表的优先级【重点】
	行内样式表优先级最高；
	外链样式表和内嵌样式表优先级取决了书写位置，后书写的优先级别高；
	
5、“对象”是很重要的，它指定了对哪些网页元素进行设置，因此，它有一个专门的名称——选择器（selector）。
6、CSS选择器的类型：
	全局选择器(通配符选择器)，
	标签选择器（元素选择器）
	id选择器
	类选择器
	伪类选择器
	群组选择器
	后代选择器（包含选择器）
	子选择器
	相邻选择器
	兄弟选择器
	属性选择器
	
	(1)全局选择器：设置所有标签使用同一样式，用*表示。
	语法：*{属性:属性值}
	如：*{margin：0;padding：0} 通配符选择器
	
	(2)标签选择器(元素选择器)：用于声明标记采用的样式。所有的标签，都可以是选择器。
	语法：元素名称{属性:属性值}
	如：p{color:red;}
	
	(3)用来为一系列标签定义相同的样式。类别选择器在页面上可以出现多次。
	语法：.类名 {属性:属性值}
	步骤：
		 a)先使用class定义类名称：如  
		  	<p class="p1">这是一个段落</p>
		 
		 b)设置并应用样式：
			  <style type="text/css">
			   .p1{属性：属性值；}
			  </style>
	
	(4)id选择器：和类选择器相似，但要注意同一id名在同一个页面中只能出现一次。
	语法：#id {属性:属性值}
	步骤：
	 	a)先使用id定义id选择器名称：如
	  		<p id="p1">这是一个段落</p>
	
	 	b)设置并应用样式：
		  <style type="text/css">
		 	  #p1{属性：属性值；}
		  </style>

	任何的HTML标签都可以有id属性。表示这个标签的名字。
	id命名规则：
	   1） 只能有字母、数字、下划线
	   2） 必须以字母开头
	   3） 不能和标签同名。比如id不能叫做body、img、a
	
	(5)伪类选择器：用伪类定义的样式并不是作用在标记上，而是作用在标记的状态上。
	语法：
	a:link {color: #FF0000}		/* 未访问的链接 */
	a:visited {color: #00FF00}	/* 已访问的链接 */
	a:hover {color: #FF00FF}	/* 鼠标移动到链接上 */
	a:active {color: #0000FF}	/* 选定的链接 */
	当超链接的四个伪类选择器联合使用时，要注意他们的属顺序。正常书序是lvha，错误的顺序有时会导致超链接的样式失效。

	【备注：】
	a:link 定义未访问过的超链接的样式，但是有些浏览器会将它解析为任何超链接，包括访问过的和未访问过的。因此，常使用a选择器来代替a:link选择器的样式。

	四种状态：（顺序不可变LVHA）

	一般 只设置 超链接 默认效果和 鼠标划过的效果：
	 a{}
	 a:hover{}
	
	(6)群组选择器：
	语法：选择器1,选择器2,选择器3 {属性:属性值}
    如：h1,h2,h3,h4,h5,h6{ color:red; font-size:30px}
	
	(7)包含选择器（后代选择器）
	语法：选择器1  选择器2   {属性:属性值}
	与子选择符不同的是，包含选择符将会命中所有符合条件的后代，包括儿子，孙子，孙子的孙子... 
	  html div{ } 
	 .header li{}
	  div span{}
	
	(8)子选择器
 	语法：选择器1 > 选择器2   {属性:属性值}   
  	p>a{}   p内标签名为a的元素样式
	
	
	(9) 相邻选择器E+F ：选择紧贴在E元素之后F元素。
	
	
	(10)兄弟选择器E~F ：选择E元素后面的所有兄弟元素F。 
	
	(11)属性选择器
       E[att] ：选择具有att属性的E元素。
       E[att="val"]：选择具有att属性且属性值等于val的E元素。 
       例如：input[type="text"]
        
7、CSS选择器优先级
	1、CSS样式表优先级
	内联样式表优先级最高；
	外链样式表和内嵌样式表优先级取决了书写位置，后书写的优先级别高；

	2、css样式优先级
	        权值：
	内联样式：                 1000
	ID选择器：	       100  	 #p2{}
	类选择器：		   10  	 .p1{}
	标签选择器：	       1  	  p{}
	全局选择器：	       0     *{}

	3、!important
	!important作用是提高指定CSS样式规则的应用优先权。
	color: blue !important;
	
	4、优先级为:
　　!important  > 行内样式表 >  id  > class  > 属性选择器  > 伪类  > 标签  >  全局
