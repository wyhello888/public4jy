一、浮动元素的使用场景：
	1、利用div浮动实现布局
	默认div都独占一行，如果希望div在同一行展示，最好的方式是设置浮动。因此div浮动是实现页面布局的必选方式；
	2、利用列表浮动实现导航
	创建水平导航栏或水平菜单时，常常使用ul，将li统一左浮动来实现；
	3、实现图片排列顶对齐
	几个高度不同的img在一排展示，默认底部对齐。如果希望顶部对齐排列，那么最好的方式就是使用浮动；
	4、实现文字饶图效果
	5、制作段落的首字母浮动于左侧效果
	
二、浮动的语法：
	1、浮动float属性
	float语法： 
	float : none | left |right
	参数值： 
	none : 　对象不浮动
	left : 　对象浮在左边
	right : 　对象浮在右边
	
三、使用浮动必须注意的细节
	1）加了浮动，一定要清除；
	2）同级元素，只要一个浮动，其它都要浮动；
	3）浮动元素尽量设置宽度；（浮动元素如果没有设置宽高，浮动后就是内容的宽高）
	4）在 CSS 中，任何元素都可以浮动。浮动元素会生成一个块级框，而不论它本身是何种元素；（浮动可以让行内元素支持宽高）
	5）假如在一行之上只有极少的空间可供浮动元素摆放，那么这个元素会跳至下一行，这个过程会持续到某一行拥有足够的空间为止；
	6）浮动块向下一行移动寻找空间时，会出现被其它浮动元素“卡住”的现象；
	7）浮动是半脱离文档流


四、如何清除浮动？
	1、设置父容器的高度（该高度要保证完全容纳所有的子元素，如果高度不够高，就没有完成清理浮动，就会导致后续的元素出现错位）
	2、父容器内部的子元素最后增加一个空元素来清除浮动，利用clear:both;
	3、父容器设置overflow:hidden，或overflow:scroll来清除浮动。（其作用是将父容器的高度进行测量，等同于给父容器设置高度）
	4、利用伪元素清除浮动，

			.clearfix:after {
				content: ""; /*设置内容为空*/
				height: 0; /*高度为0*/				
				display: block; /*转为块级元素*/
				visibility: hidden; /*将元素隐藏*/
				clear: both; /*清除浮动		*/
			}			
			.clearfix {
				zoom: 1;/*为了兼容IE*/	
			}
